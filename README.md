# ChaosMock

**API Chaos Engineering Sandbox**

ChaosMock is a developer tool designed to help frontend teams build resilient applications. It allows developers to create mock API endpoints that intentionally inject "chaos" (such as random latency, HTTP 500 errors, or malformed JSON data).

## Tech Stack

**This project is built using this architecture:**

* **Frontend:** Flutter
* **Backend:** Kotlin & Spring Boot
* **Database:** PostgreSQL
* **Infrastructure:** Docker, Kubernetes (K8s), AWS
